Search product by product ID and location ID

Search on this page by product ID. Seach by Product Id will return all the products with the entered snd display product details and the locations where the products are.

Seach by Location Id will return all the products in the location with the ID.

Enter the product id of the item you want to seach. Product Id is a value between 1 and 1000.

Enter the Location id of the location to view items at that location. Location Id is a value between 1 and 1000.
